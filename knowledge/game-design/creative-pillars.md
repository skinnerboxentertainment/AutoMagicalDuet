# Creative Pillars

Creative pillars are non-negotiable decision filters. They must be few, falsifiable, and useful when departments disagree.

## Patterns

- Keep 3-5 pillars maximum.
- Make each pillar specific enough to resolve a real design dispute.
- Define a design test for every pillar.
- Include anti-pillars that state what the game is not.
- Evaluate decisions by core fantasy, pillar alignment, target aesthetics, coherence, positioning, and feasibility.
- When cutting scope, protect features that are the pillars, simplify features near pillars, and cut features far from pillars first.

## Anti-patterns

- Vague pillars such as "fun gameplay" or "high quality."
- Pillars that constrain only one department.
- Scope cuts that remove the feature that carries the core fantasy.
- Creative decisions that improve one pillar while silently violating another.
